# 📝 패치 노트

## 🔄 Version 1.1.0 - RAG 검색 개선 (2024-12-19)

### 🐛 버그 수정

#### 용어 검색 실패 문제 해결
- **문제**: 공백이 포함된 용어 검색 시 매칭 실패
  - 예: 'p2p 금융업' 검색 시 'P2P금융업'을 찾지 못함
- **원인**: 정확 매칭 로직이 공백 차이를 고려하지 않음
- **해결**: 공백 정규화 기능 추가

### ✨ 새로운 기능

#### 1. 공백 정규화 검색
- 용어 검색 시 공백(일반 공백, 전각 공백)을 자동으로 제거하여 비교
- 대소문자 무시와 함께 공백 무시 비교 지원
- **예시**:
  - 입력: `'p2p 금융업'` → 정규화: `'p2p금융업'`
  - 저장: `'P2P금융업'` → 정규화: `'p2p금융업'`
  - ✅ 매칭 성공!

#### 2. 벡터 검색 Fallback
- 정확 매칭 실패 시 자동으로 의미 기반 벡터 검색 수행
- 유사한 용어를 찾아 관련 정보 제공
- 검색 정확도 향상

### 🔧 기술적 개선

#### 메타데이터 확장
- `term_normalized` 필드 추가: 공백 제거된 정규화 용어 저장
- 검색 성능 최적화를 위한 사전 정규화

#### 검색 로직 개선
```
기존: 정확 매칭 (대소문자만 무시)
     ↓ 실패
     → 기본 사전 사용

개선: 정확 매칭 (공백/대소문자 무시)
     ↓ 실패
     → 벡터 검색 (의미 기반)
     ↓ 실패
     → 기본 사전 사용
```

### 📊 변경된 파일

import streamlit as st

# 세션 상태 초기화

    del st.session_state.rag_initialized

# 재초기화
ensure_financial_terms()
