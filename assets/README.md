# Assets 폴더

이 폴더는 프로젝트에서 사용하는 이미지 및 기타 리소스 파일을 저장하는 곳입니다.

## 알부엉 아바타 이미지

챗봇 아바타로 사용할 알부엉 이미지를 다음 파일명 중 하나로 저장하세요:

- `albwoong.png` (권장)
- `albwoong.jpg`
- `albwoong.svg`
- `owl.png`
- `owl.jpg`

이미지를 저장하면 챗봇 메시지 왼쪽에 자동으로 표시됩니다.

## 이미지 권장 사양

- **크기**: 200x200px 이상 (정사각형 권장)
- **형식**: PNG (투명 배경) 또는 SVG
- **비율**: 1:1 (정사각형)
